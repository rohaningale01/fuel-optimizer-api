# Fuel Optimization API

## Features
- Route calculation
- Cheapest fuel stops
- Total fuel cost

## Endpoint
POST /api/optimize-route/

## Example Input
{
  "start": [-74.0060, 40.7128],
  "end": [-118.2437, 34.0522]
}

## API Key Setup

Replace `YOUR_API_KEY` in route_service.py with your OpenRouteService API key.
